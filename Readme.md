# Laundry Services Website

A modern and responsive hero section website for professional laundry services. This project showcases a clean design with a call-to-action for booking laundry services.

## Project Structure

```
# Main HTML file
# Stylesheet
# Project documentation
# Navigation logo
# Hero section image
```

##  Features
- **navbar section**
    It contains logo,home,services,about us,contact us with class name nav-links,and have username section for user details.

- **Hero Section**
    It contains the main attraction of website with heading and some intro of laundry wallah with a image. It also have a book service today button.

- **Modern Styling**
